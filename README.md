# AutoIndex

Final polished AutoIndex website built with React + TypeScript + Tailwind CSS.

## Features

- Parts valuation workflow with market range simulation
- Multi-vendor marketplace with filtering and sorting
- Listing detail, saved items, and cart flows
- Marketplace analysis panel (MVP simulated intelligence)
- Vendor onboarding flow (simulated)
- Responsive layout for desktop and mobile
- Local persistence for cart, saved items, and valuations

## Run locally

Requires Node.js 20+.

```bash
npm install
npm run dev
```

Open the local URL shown by Vite (typically `http://localhost:5173`).

## Build

```bash
npm run build
npm run preview
```

## Tech stack

- React
- TypeScript
- Vite
- Tailwind CSS
- Lucide React
